Omega Five Theme Information
============================
Omega Five is a complete rewrite of Omega that brings back (and vastly improves) the user friendly 
layout interface that Omega was always known for. Responsive theming should be available to all 
site builders, themers and developers. 

All the amazing things that Omega provides can be disabled via the interface, or via the theme 
settings. By default, for most users new to Omega, and looking for a simple, effective, responsive 
layout for their new project will use the provided grid system, Omega.gs (https://github.com/ThemeGeeks/omega.gs).

Omega.gs is a SCSS grid system based on 960.gs, semantic.gs and some custom elements to provide a 
very simple method of creating SCSS layouts. When editing the layout via the administrative interface, 
after saving changes, the option exists to rewrite the SCSS and compiled CSS versions of omega-layout.scss/css. 
Zero performance load is lost to front end visitors as the plain old CSS file is included to control all 
CSS related to the region layout.

Should you want to use your own grid system, or provide and even more advanced layout in your Omega Five 
subtheme, you can easily turn off the Omega.gs integration via the interface, or theme settings, and provide 
your own layout CSS in whatever manner you see fit.
